Що таке Періодичний Календар?

Періодичний Календар це програма, яка дозволяє:
  • відслідковувати менструальні цикли
  • передбачати періоди плідності, коли ймовірність завагітніти є найвищою
  • передбачати "безпечні" дні коли ймовірність завагітніти є найменшою
  • передбачати дату наступної менструації
  • занотовувати періоди вагітності
  • вираховувати номер тижня вагітності
  • передбачати дату пологів
  • вести особисті нотатки

На даний момент для визначення плідних днів використовується виключно календарний метод, який не є надійним. Для природного планування сім'ї необхідно використовувати ще й інші методи, які даватимуть більшу надійність.

В майбутніх версіях планується додати підтримку Температурного методу, який в сукупності з календарним дає більш точні прогнози. Однак цей метод вимагає більше зусиль для його практикування (щоденні вимірювання температури) в той час як для календарного методу достатньо просто не забути занести в програму дату початку менструального циклу. Ефективність температурного методу оцінюють десь на рівні 80-90%. Комбінування більш ніж одного метода завжди збільшує надійність прогнозів вцілому.

Інші методи, що базуються на самоспостереженні, досить важко автоматизувати на комп'ютері.
