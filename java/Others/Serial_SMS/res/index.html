Simple SMS Client in Java for serial connection
      ***************************************************************

        Created 2001 by Stefan Moscibroda

  • Purpose 

    This Java SMS Client aims to support the communication between a computer
    and a mobile device via a serial connection. This serial connection can either be a
    serial cable or a infrared connection (IRDA with virtual COM port). Serial drivers
    for Windows are included in this package as well as an installation guide.

    The tool allows sending, reading and writing of SMS in both transmit and
    received format, as well as the sending of ordinary AT commands to the mobile. 
    The tool is using the PDU SMS format which is supported by most GSM mobiles.

  • Functionality of the SMS tool / classes

    - Sending and receiving SMS (also time information icluded)

    - Writing and Reading SMS from Mobile/SIM storage, Deleting SMS 
    (support of both receiving and transmitting format)

    - Direct support of AT command: 
      Can be used to e.g. read out Phone book entries, IMEI of the mobile, etc.

    For the SMS functionality make sure that the mobile is set to SMS PDU mode.
    To set the mobile to PDU mode, type the AT commad "AT+CMGF=0"


   The functionality of this SMS Client can be easily integrated into your own 
   applications. Feel free to make your extensions!
